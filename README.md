# MathMaster - Interactive Mathematics Learning Platform

🎓 A premium, interactive web-based platform for learning mathematics through visual animations and gamification.

## 🌟 Features

### 1. **Interactive Visualizers**
   - **Multiplication Visualizer**: See multiplication as a grid of colored squares
   - **Division Visualizer**: Understand division through grouped items
   - **Addition Visualizer**: Watch numbers combine with animated bars
   - **Subtraction Visualizer**: See subtraction as removing items from a bar

### 2. **Real-Time Animations**
   - Pop-in animations for grid items
   - Smooth bar growth animations
   - Pulsing result displays
   - Hover effects on all interactive elements

### 3. **Quiz Mode**
   - Generated math questions across all operations
   - Multiple choice answers
   - Real-time feedback (correct/wrong animations)
   - Progress tracking with statistics
   - Automatic difficulty based on performance

### 4. **Progress Tracking**
   - Track correct and incorrect answers
   - View accuracy percentage
   - Weekly performance chart
   - Achievement system
   - Learning streak counter

### 5. **Premium Design**
   - Modern gradient backgrounds
   - Smooth scrolling navigation
   - Responsive grid layouts
   - Glass-morphism effects
   - Professional color scheme

## 🎨 Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Advanced animations and gradients
  - CSS Grid & Flexbox
  - Keyframe animations
  - CSS variables for theming
  - Backdrop filters
- **Vanilla JavaScript** - No frameworks
  - Dynamic visualization generation
  - Event handling
  - Intersection Observer API
  - DOM manipulation

## 🚀 How to Use

1. **Multiplication Learning**
   - Input two numbers (1-12)
   - Click "Visualize" to see the multiplication as a grid
   - Watch colored squares animate and see the result

2. **Division Learning**
   - Enter a dividend and divisor
   - Watch items group into equal sets
   - Understand how many items are in each group

3. **Addition Learning**
   - Enter two numbers
   - See bars representing each number
   - Watch them combine with smooth animation

4. **Subtraction Learning**
   - Input numbers to subtract
   - See the minuend and subtrahend as bars
   - Watch items be removed to show the difference

5. **Quiz Mode**
   - Answer automatically generated questions
   - Get instant feedback
   - Track your progress and improve

## 🛠️ Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #ec4899;
    /* ... */
}
```

### Adjust Difficulty Ranges
Modify the input `max` and `min` attributes in `index.html`

### Add More Operations
Duplicate a tab section and create new visualization function
