## 📊 Key Components

### Multiplication Grid
- Creates visual grid representation
- Each cell is an animated square
- Helps understand multiplication as repeated groups

### Division Groups
- Shows items divided into equal groups
- Circular items with group spacing
- Makes division concept tangible

### Addition/Subtraction Bars
- Visual bar representation of numbers
- Color-coded for different operations
- Shows combined or remaining values

### Quiz System
- Random question generation
- Smart answer shuffling
- Correct/wrong animations
- Real-time statistics update

## 🛠️ Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #ec4899;
    /* ... */
}
```

### Adjust Difficulty Ranges
Modify the input `max` and `min` attributes in `index.html`

### Add More Operations
Duplicate a tab section and create new visualization function
